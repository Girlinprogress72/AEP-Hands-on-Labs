Instructions:
-----------------
1. Copy and paste this SQL command

```sql
SELECT concat(placeContext.geo.stateProvince, ' - ', placeContext.geo.city) AS state_city, 
       Count(timestamp) AS Count
FROM   media_demo_data_postvalues
WHERE  _acp_year = 2019
       AND _acp_month = 11
       AND _acp_day = 03
GROUP BY state_city
ORDER BY Count DESC
LIMIT  10;
```
